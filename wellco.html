New Here? Start Simple ✨

If you’re building a basic routine, I’d start with a few foundation products first and keep it easy. These are great starting points for a simple daily stack:

Many of these products fit best into a repeatable daily rhythm. Here’s one simple way to think about building a wellness stack:
